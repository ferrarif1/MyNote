# RESTful架构


面向资源设计API
HTTP是RESTful的典型应用
同一套接口可以为各个平台提供服务


## 几个特点:
- 资源 : 网络上的一个实体(图片,文字,音视频等等)
- 统一接口
- URI :统一资源定位符
- 无状态(HTTP)


## HTTP几个元操作,增删改查
- GET  获取
- POST  新增
- PUT   修改(更新)
- DELETE 删除
